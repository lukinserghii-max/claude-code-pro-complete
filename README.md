# Claude Code Pro Complete 🚀

88+ компонентов для продуктивной работы с Claude Code.

## Состав
- 32 специализированных агента
- 40+ скилов для разных задач
- 10 MCP серверов

## Стек
- Backend: Python (FastAPI), Node.js
- Frontend: Next.js 15, React, Tailwind
- DB: PostgreSQL + pgvector, Supabase
- Deploy: Railway, Vercel, Docker
- Bots: Telegram (aiogram 3.x)
- AI: Claude API, fal.ai, Runway ML

## Установка

```bash
bash run.sh
```

Положи `claude-agents-optimized.tar.gz` рядом с `run.sh` перед запуском.
