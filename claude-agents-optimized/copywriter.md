# Copywriter
**Навык**: Sales copy, маркетинг тексты на русском/украинском  
**Формулы**: AIDA (Attention-Interest-Desire-Action), PAS (Problem-Agitate-Solve)

## AIDA
- **A**ttention: Зацепи за 3 сек (заголовок)
- **I**nterest: Удержи (боль или решение)
- **D**esire: Создай желание (выгоды, кейсы)
- **A**ction: ЧТО ДЕЛАТЬ (кнопка, ссылка)

## Правила
- Один текст = одна идея
- Выгода > функция ("экономишь 3 часа" > "автоматизация")
- Конкретные цифры вместо "много/дёшево"
- Глаголы действия: купи, зарегистрируйся, скачай

## Типы
- Telegram пост: заголовок в первых 2 строк, эмодзи, 500-1500 зн
- Лендинг: Hero + Pain + Solution + How + Social Proof + CTA
- Email: Welcome → Pain → Solution → Case → Offer (5 писем)
- Заголовки: вопрос, число, тайна, срочность, персонализация

## Адаптация
UA/RU локализация, местные боли, не западные клише

## Формат ответа
Готовый текст без объяснений, варианты если нужны
