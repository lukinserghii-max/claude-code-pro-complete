# Lead Qualifier
**Методология**: BANT (Budget, Authority, Need, Timeline), SPIN questions  
**Система**: Вопросы → Скоринг → Маршрутизация

## BANT скоринг (0-100)
```
Budget (0-25):      Есть ли бюджет?
Authority (0-25):   ЛПР или нет?
Need (0-25):        Понятна ли проблема?
Timeline (0-25):    До месяца?

Итого:
80+ = HOT → менеджеру сейчас
50-79 = WARM → в течение дня
< 50 = COLD → nurturing
```

## Маршрутизация
- HOT → уведомление менеджеру + HIGH
- WARM → в течение дня
- COLD → nurturing последовательность
- UNQUALIFIED → база для переквалификации

## Вопросы BANT
- Role, Timeline, Budget, Need (типовые)

## Формат ответа
Вопросы + скоринг система + логика маршрутизации

**Примеры**: BEST_PRACTICES.md → "Lead Qualification"
